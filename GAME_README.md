## Installation & Setup

### Step 1: Install Dependencies

```bash
pip install -r game_requirements.txt
```

This installs:
- `pygame` - Game engine
- `requests` - For downloading assets from PokeAPI

### Step 2: Download Game Assets

**IMPORTANT:** Before playing, you must download Pokemon data and sprites:

```bash
python download_assets.py
```

This will:
- Download sprites for starter Pokemon (Bulbasaur, Charmander, Squirtle)
- Download sprites for wild Pokemon (Pidgey, Rattata, Caterpie, Weedle, etc.)
- Fetch Pokemon stats, moves, and type data from PokeAPI
- Save everything to `game_assets/` folder

The download process takes 2-5 minutes depending on your internet speed.

### Step 3: Run the Game

```bash
python pokemon_game.py
```

## How to Play

### Starting the Game

1. **Title Screen** - Press SPACE to continue
2. **Starter Selection** - Click on one of three starter Pokemon to begin your adventure
3. **Overworld** - You'll spawn in a grassy field ready to explore

### Controls

#### Overworld
- **Arrow Keys** - Move your character (red square)
- **Walk through dark green grass** - Trigger random wild Pokemon encounters (5% chance per tile)

#### Battle
- **Mouse Click** - Select moves or actions
- **Move Buttons** - Choose one of your Pokemon's 4 moves to attack
- **Run Away Button** - Attempt to flee (50% success rate)
- **ESC Key** - Return to overworld after battle ends

### Battle System

1. **Encounter** - Walk through grass to encounter wild Pokemon
2. **Your Turn** - Choose a move to attack or try to run
3. **Enemy Turn** - Wild Pokemon attacks back automatically
4. **Victory** - Defeat the wild Pokemon to gain 50 EXP
5. **Defeat** - If your Pokemon faints, the battle ends
6. **Escape** - Successfully running ends the battle

### Pokemon Stats

Each Pokemon has:
- **HP (Hit Points)** - Health remaining
- **Attack** - Physical damage power
- **Defense** - Damage resistance
- **Speed** - Turn order (not fully implemented)
- **Level** - Power multiplier
- **Experience** - Progress to next level

### Leveling Up

- Win battles to gain 50 EXP
- Level up when EXP bar fills
- Stats increase on level up:
  - HP: +5
  - Attack: +2
  - Defense: +2
  - Speed: +2

## Troubleshooting

### "Game assets not found" Error
**Solution:** Run `python download_assets.py` first

### Sprites Not Loading
**Solution:** 
1. Delete `game_assets/` folder
2. Re-run `python download_assets.py`
3. Check your internet connection

### Slow Performance
**Solution:** 
- Close other applications
- Reduce encounter rate in code (line 253 in pokemon_game.py)
- Update graphics drivers

### No Wild Encounters
**Solution:** 
- Walk through the **dark green grass tiles**
- Keep moving - 5% chance per tile
- Wait for encounter cooldown (2 seconds between encounters)

### Battle Stuck
**Solution:** 
- Wait for message timer (2 seconds per message)
- Click a move button when it's your turn
- Press ESC after battle ends

Edit line 253 in `pokemon_game.py`:
```python
if random.random() < 0.05:  # Change 0.05 to desired rate (0.1 = 10%)
```

### Modify Damage Formula
Edit the `calculate_damage` method in `BattleSystem` class

Edit lines 133-134 in `pokemon_game.py`:
```python
self.world_width = 1920  # Change width
self.world_height = 1440  # Change height
```
